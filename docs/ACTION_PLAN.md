## 📋 각 화면 필수 요소 (와이어프레임에 포함할 것)

### 홈 화면 (예원)
- 검색바
- 카테고리 필터 칩 (서브웨이/하이디라오/편의점)
- 조합 카드 리스트
  - 이미지 썸네일
  - 제목
  - 간단 설명
  - 좋아요 수
  - 작성자

### 조합 상세 화면 (예원)
- 이미지 (상단)
- 제목
- 작성자 정보
- 재료/비율 리스트
- 만드는 방법 (스텝)
- 좋아요 버튼
- 댓글 섹션

### 조합 등록 화면 (일혁)
- 이미지 업로드 버튼
- 제목 입력
- 카테고리 선택
- 재료 입력 (태그 형태)
- 설명 입력
- 등록 버튼
