# 역할 분담

## 예원
- 홈 화면 (HomeScreen)
- 조합 상세 화면 (DetailScreen)
- 관련 ViewModel, Repository
- 홈 화면 및 조합 상세 화면의 API 연동
- 홈 화면 및 조합 상세 화면의 디자인 적용

## 일혁

### 인증 관련
- 로그인 화면 (LoginScreen)
- 회원가입 화면 (RegistrationScreen)
- 아이디 찾기 화면
- 비밀번호 찾기 화면
- 소셜 로그인 API 연동

### 조합 관련
- 조합 등록 화면 (CreateCombinationScreen)
- 이미지 서버 업로드
- 조합 등록 API 연동

### 사용자 관련
- 마이페이지 (MyScreen)
- 프로필 수정 화면 (EditProfileScreen)
- 주문 내역 화면
- 마이페이지 및 프로필 수정 API 연동

### 기타 화면
- 검색 화면 (SearchScreen)
- 스플래시/온보딩 화면 (SplashScreen)
- 설정 화면
- 공지사항 화면
- 이벤트 화면
- FAQ 화면
- 1:1 문의 화면
- 앱 설정 화면
- 개인정보 처리방침 화면
- 서비스 이용약관 화면

### 공통 작업
- 공통 컴포넌트 (CombinationCard, CategoryChip, SearchBar, TopAppLogoBar 등)
- 폰트 설정 (Pretendard 폰트 적용)
- API 연동 (일혁 담당 화면들)
- 에러 처리 개선
- 로딩 상태 개선
- 검색 기능 고도화
